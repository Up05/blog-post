Komentarai

Komentarai yra tiesiog tekstas kitiems žmonėms, kompiliatoriui, arba kompiuteriui jie visai nerūpi ir tai juos tiesiog ištrina.

Komentarai dažniausia rašomi su dvejais pasviraisiais brūkšniais //, arba # ir tada pasibaigia su eilutės pabaiga. Bet jie gali būti rašomi ir /* komentaras */ ir kai kuriose kalbose dar kitais būdais, bet šito tūrėtų užtekti.

// paverčia milisekundes į valandas.
function mil2val(mil){
  // Žmonėms, kurie bijo dalybos: kompiliatoriai iš karto padaro tokią dalyba, taigi: let val = mil / 3,600,000; (plius/minus)
  let val = mil / 1000 / 60 / 60;
  // Math.floor() numeta visus skaičius, kurie yra po kablelio (apvalina žemyn).
  val = Math.floor(val);
  /* 
  Ši eilutė dabar nieko nedaro, nes ji paversta komentaru, bet kitaip būtų kiek valandų dienoje. 
  Procentų ženklas neturi NIEKO bendro su procentais čia, tai yra liekana (dalyba, kai nenori turėti skaičių po kablelio)
  */
  // val %= 24;
  return val;
}

Tiek daug komentarų nereikėtų rašyti, nes žmonės, kurie skaitys tavo kodą darbe turėtų žinoti kas yra Math.floor() ir liekanos. Be to skaityti kas antrą eilutę su labai skirtingais eilučių ilgiais yra sunku. Čia ta pati funkcija be komentarų.
