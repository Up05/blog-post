Funkcijos

Funkcija yra operacija, arba operacijų seka, kurią tu norėtum suskaičiuoti, arba padaryti daugelį kartų, jos nėra būtinos, bet be jų reikėtų kopijuoti LABAI daug kodo (arba šokinėti į kažkur ir atgal). Funkcijos gali paimti parametrus, arba argumentus, kažką paveikti ir atiduoti kažkokią reikšmę.

Funkcijos, pavyzdžiui C kalboje rašomos: reikšmės-tipas pavadinimas (arg-tipas arg-pavadinimas) {bet-kas; return reikšmė;}.

int atimti(int a, int b){
  return a - b;
}

Kaip ir kintamieji, skirtingose kalbose funkcijos rašomos truputi kitaip, pavyzdžiui Javascript pakeičia reikšmės tipą į function žodį ir tiesiog išmeta argumentų tipus, taigi:

function atimti(a, b)

Rust seka savo savotišką rašymą:

fn atimti(a:i32, b:i32) -> i32 {a - b}
Rust, kaip C ir Java turi turėti kabletaškius, taigi kai mes nepadedame jų funkcijose, Rust interpretuoja tai, kaip return ...;, arba funkcijos reikšme. { a + b } yra taspats, kaip { return a + b; }
