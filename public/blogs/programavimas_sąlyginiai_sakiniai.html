Sąlyginiai sakiniai

Sąlyginiai sakiniai atlieka kažkokį jiems duotą kodą, jei jiems duota sąlyga yra teisinga. Jie dažniausiai rašomi su if, kas išvertus į lietuvių kalbą reikškia jei. Pavyzdys su C: 

int a = 5;
if(a == 5)
  a = 6;

int a = 5;
if(a == 5){
  a = 6;
}

{}, arba riestiniai skliaustai, šiuo atveju nėra būtini, nes if() turi tik vieną eilutę.
Čia a bus 5, kompiuteris pasižiūrės ar a yra penki ir, jei a bus lygu penkiems jis pakeis a reikšmę į 6.

Sąlyginiai sakiniai taip pat turi else žodį, kuris išvertus į lietuvių kalbą, turbūt, reikškia o. Jei a yra lygu šešiems daryk taip, o, jei ne daryk kitaip.

int a = 5;
if(a == 5)
  a = 6;
else
  puts("a NĖRA lygu 5"); 
  // puts(), yra funkcija, kuri tiesiog išmeta tekstą į konsolę, 
  // šiuo atveju nebūtina žinoti ką ji daro ir kaip.

Beto sąlyginiai sakiniai turi jungtukus ir kelis lygumo operatorius.

Operatorius Paaiškinimas
! NE
&& IR
|| IR/ARBA
Operatorius Paaiškinimas
== LYGU
> DAUGIAU NEI
< MAŽIAU NEI
>= ir <= DAUGIAU/MAŽIAU, ARBA LYGU
!= NELYGU
 
int a = 5;
if((a == 5 || a == 6) && a != 4)
  a = 6;

Jei a lygu penki, arba šeši, bet ne 4, a bus šeši.
Beje čia sąlygos nėra labai protingos, nes, pavyzdžiui, šioje sąlygoje, nei 5, nei 6 nelygu 4, taigi && a != 4 dalis yra visai nereikalinga.
